## **关于服务器**

### - OF服由繁星服（小狗服）分裂而来，目前的OF服与繁星服无较大关联。繁星服约在2023年暑假建立，腐竹繁星（小狗），目前OF服腐竹初（TheAchu）担任建筑，分成五五开。2024年暑假，繁星服开服，开服费用五五开。不出一个月，因信任危机与分成问题出现争执。8月，初另起炉灶，将原先在繁星服所建地图归回己有，建立OF服。
